## Цель исследования
Сравнить качество решения прикладных задач горнорудной отрасли с помощью зарубежного (ChatGPT-4o) и отечественного (GigaChat-Pro) ИИ-ассистентов.

# Выбранные кейсы

## Кейс 1: Диагностика неисправностей насосного оборудования
- Описание: Моделирование ситуации с падением давления в маслосистеме и повышенной вибрацией насоса ЦНС-180
- Цель: Проверить способность ботов к техническому анализу, пониманию физики процессов и предложению алгоритмов диагностики

### Запрос для ботов:
Сценарий: Насос ЦНС-180 (центробежный нефтяной секционный) подает масло с пониженным давлением (норма 6 атм, факт 3.2 атм), при этом температура корпуса повышена (65°C при норме 45°C). Датчик вибрации показывает значения выше нормы (7.5 мм/с при норме до 4.5 мм/с).

Задача:

1. Предложи пошаговый алгоритм диагностики причины неисправности
2. Перечисли возможные причины, ссылаясь на физику процессов
3. Составь чек-лист для ремонтной бригады перед остановкой оборудования


### Результаты тестирования:

#### ChatGPT-4o (Зарубежный бот):

Достоинства:
- Подробный пошаговый алгоритм с временными интервалами
- Глубокое объяснение физических процессов (кавитация, гидроудары, тепловое расширение)
- Структурированный ответ с таблицами и схемами
- Учет современных методов диагностики (спектральный анализ вибрации)

Недостатки:
- Незнание российских стандартов и ГОСТов
- Отсутствие специфики отечественного оборудования
- Рекомендации ссылаются на зарубежные стандарты (API 610 вместо ГОСТ 32718-2014)

#### GigaChat-Pro (Отечественный бот):

Достоинства:
- Точное соответствие российским нормативам (ссылки на ПБ 14-586-03, РД 08-622-03)
- Знание типовых неисправностей для насосов ЦНС
- Практические рекомендации с учетом климатических условий
- Четкое соблюдение правил охраны труда РФ

Недостатки:
- Менее детальное описание физических процессов
- Отсутствие современных методов диагностики
- Более консервативный подход к решению

### Сравнительная таблица результатов Кейса 1:

| Критерий | ChatGPT-4o | GigaChat-Pro |
|----------|------------|--------------|
| Полнота ответа | 9/10 | 8/10 |

## Кейс 2: Извлечение требований охраны труда из инструкции

- Описание: Анализ фрагмента инструкции по охране труда при взрывных работах
- Цель: Проверить способность ботов к извлечению структурированной информации из нормативных документов

### Контекст (загруженный текст инструкции):
ИНСТРУКЦИЯ ПО ОХРАНЕ ТРУДА ПРИ ПРОВЕДЕНИИ ВЗРЫВНЫХ РАБОТ

1. Требования охраны труда при подготовке взрывных работ
   3.1. Доставка взрывчатых материалов к месту работ должна осуществляться специальным транспортом. Запрещается перевозка ВМ вместе с людьми.
   3.2. Заряжание шнуров и скважин производится механизированным способом или вручную. При ручном заряжании запрещается применять металлические трамбовки.
   3.3. Расстояние от места взрыва до охраняемых объектов должно быть не менее 200 м для зданий и 300 м для ЛЭП.
2. Требования безопасности в аварийных ситуациях
   4.1. При обнаружении отказа (неполного взрыва) работы прекращаются, выставляется оцепление в радиусе не менее 500 м.
   4.2. Ликвидация отказавших зарядов производится по проекту, утвержденному техническим руководителем.
   4.3. Запрещается извлекать отказавший заряд, бурить рядом с ним дополнительные шпуры.
3. Средства индивидуальной защиты
   5.1. Взрывники должны быть обеспечены: каской защитной (ГОСТ 12.4.128), противошумными наушниками (ГОСТ 12.4.275), спецобувью с антистатическими свойствами.
   5.2. При работе с сыпучими ВВ обязательно применение респираторов типа ШБ-1.


### Запрос для ботов:
На основе предоставленного документа:

1.Выяви все требования к средствам индивидуальной защиты (СИЗ)
2. Сформулируй краткую памятку для взрывника при обнаружении отказавшего заряда
3. Составь чек-лист действий для сотрудника при возникновении аварийной ситуации


### Результаты тестирования:

#### ChatGPT-4o (Зарубежный бот):

Достоинства:
- Отличное структурирование информации
- Создание удобной памятки в формате "делай/не делай"
- Логичное дополнение общими принципами безопасности

Недостатки:
- Неверная интерпретация ГОСТов (путаница с международными стандартами)
- Добавление требований, отсутствующих в исходном документе (галлюцинации)
- Игнорирование специфических российских терминов

#### GigaChat-Pro (Отечественный бот):

Достоинства:
- Точное извлечение всех требований без добавлений
- Правильная интерпретация ГОСТов и нормативов
- Понимание специфики российских взрывных работ
- Соблюдение юридической корректности формулировок

Недостатки:
- Менее структурированный ответ
- Бюрократический стиль изложения
- Отсутствие визуальных элементов (таблиц, схем)

# Методология

1. Сформулирован единый запрос для каждого кейса
2. Запросы отправлены обоим ботам в новом чате (один временной контекст)
3. Результаты сохранены в папку results/ (скриншоты и текстовые логи)
4. Проведен сравнительный анализ по критериям: полнота ответа, соответствие отраслевой специфике, наличие "галлюцинаций", удобство восприятия

# Основные выводы

1. Зарубежный бот (ChatGPT-4o) продемонстрировал:
   - Превосходную структурированность ответов и удобство восприятия
   - Глубокое понимание физических процессов
   - Склонность к "галлюцинациям" и добавлению несуществующих требований
   - Незнание российской нормативной базы

2. Отечественный бот (GigaChat-Pro) показал:
   - Высокую точность в работе с российскими нормативами и ГОСТами
   - Отсутствие ложных фактов
   - Понимание специфики отечественного оборудования
   - Менее удобный для восприятия формат ответов

3. Практические рекомендации:
   - Для задач, связанных с российскими нормативами и документацией, предпочтительнее использовать GigaChat-Pro
   - Для технического анализа, диагностики и обучения персонала эффективнее ChatGPT-4o
   - Оптимальный подход - комбинированное использование обоих ботов с последующей проверкой специалистом

4. Ограничения исследования:
   - Тестирование проводилось на ограниченном наборе кейсов
   - Использовались веб-версии без тонкой настройки параметров
   - Результаты могут варьироваться в зависимости от формулировки запросов

# Приложение: Примеры ответов ботов

## Кейс 1: Фрагмент ответа ChatGPT-4o
АЛГОРИТМ ДИАГНОСТИКИ НАСОСА ЦНС-180:ШАГ 1 (0-15 мин): Проверка первичных параметров

· Сравнить показания манометров на входе и выходе
· Измерить ток электродвигателя (норма до 45А)
· Проверить уровень масла в подшипниках

Физика процесса: Падение давления при росте температуры указывает на увеличение внутренних утечек. Увеличение вибрации + рост температуры = вероятный износ подшипников или дисбаланс ротора...


## Кейс 1: Фрагмент ответа GigaChat-Pro
ПОРЯДОК ДЕЙСТВИЙ ПРИ НЕИСПРАВНОСТИ НАСОСА ЦНС-180:

В соответствии с РД 08-622-03:

1. Произвести визуальный осмотр насосного агрегата
2. Проверить соответствие параметров паспортным данным
3. При подтверждении неисправности оформить наряд-допуск на ремонт

Возможные причины (согласно типовым неисправностям ЦНС):

· Износ рабочего колеса (увеличение зазоров)
· Засорение проточной части
· Износ подшипников скольжения
· Нарушение соосности валов
